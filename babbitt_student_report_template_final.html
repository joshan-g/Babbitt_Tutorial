5.2 Experimental Variograms

Describe lag spacing, number of lags, directional setup, tolerance angles, and whether omnidirectional and directional variograms were used. Present this for both Cu and Ni.

Student Instruction Box — Section 5.2
What this section should do.

This section explains how you built the experimental variogram from the data. The purpose is to show how grade similarity changes with distance and direction. You are not fitting the model yet. You are first describing what the data itself says about spatial continuity.

You should explain the following clearly:

  • Lag definition: State the lag distance and the number of lags. Explain why that lag spacing was reasonable for the sample spacing and the scale of the deposit.
  • Pair selection: Explain that each variogram point is based on sample pairs separated by a target distance interval. If relevant, mention that more pairs generally make the estimate more stable.
  • Directional analysis: State whether you computed an omnidirectional variogram, directional variograms, or both. Explain why direction matters in geologic materials and what you were checking for.
  • Tolerances and bandwidth: Report angular tolerance and bandwidth if used. Explain that these settings control which pairs are accepted into each directional variogram.
  • Main observed features: Describe what you see on the plot. Discuss whether the curve shows a nugget, where it starts to level off, whether a sill is visible, and whether continuity changes with direction.

Do not just insert the variogram figure. You must interpret it. For example, if one direction has a longer range than another, explain that this suggests stronger continuity in that direction. If the first variogram point is high, explain that this may indicate a strong nugget effect, short-scale variability, or measurement noise.

A strong response in this section answers these questions: What distances were tested? How were the pairs selected? What does the experimental variogram reveal about spatial continuity? Is there evidence of anisotropy?

5.3 Variogram Model Fitting

State the selected variogram model type(s) and report nugget, sill, and range values for both Cu and Ni.

Student Instruction Box — Section 5.3
What this section should do.

This section explains how you converted the experimental variogram into a mathematical variogram model that can be used in kriging. The experimental points alone are not enough. Kriging requires a valid continuous model.

You should explain the following clearly:

  • Model type: State whether you used a spherical, exponential, Gaussian, or nested model. Explain why that model was appropriate for the shape of the experimental variogram.
  • Nugget: Report the nugget value and explain what it represents in this study. This may reflect short-scale variability, sampling error, or measurement uncertainty.
  • Sill: Report the sill and explain how it relates to overall variability.
  • Range: Report the range and explain what it means physically. The range is the approximate distance over which samples remain spatially correlated.
  • Nested structures: If you used more than one structure, explain why. For example, a short-range structure may capture local continuity while a second structure may capture broader-scale continuity.
  • Anisotropy: If anisotropy was modeled, explain how it was incorporated. Report the directional ranges or anisotropy ratio and interpret them geologically.

Do not describe this section as curve fitting only. This section must connect the model parameters to geologic and spatial meaning. A high nugget, short range, or long range should all be interpreted in words.

A strong response in this section answers these questions: Why was this model chosen? How well does it match the experimental variogram? What do the nugget, sill, and range imply about the deposit? How will this model affect kriging?
